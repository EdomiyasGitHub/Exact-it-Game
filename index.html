How to Play NumPos

NumPos is a game that lets you accurately guess a randomly computed number by the game engine. There is a non-repeating 4-digit secret number (from 1023 to 9876) that you have to guess in order to win. There are a few game logics applied in order to make that possible.


Game Logic

Below your input field, you can see a table with three columns. Here is each column's purpose:


Attempt

Displays all your previous guesses.


N

N stands for Number and it displays how many numbers you guessed right on each attempt.


P

P stands for Position and it displays how many of the numbers you guessed are in the right position.


Example

If the secret number is 1234, and your guess is 9246:
Attempt = 9246
N = 2 (because you correctly guessed the numbers 2 and 4)
P = 1 (because your guess' 2 is at the exact position of the secret number's 2)


You win the game when both N and P are equal to 4 (which means you guessed 4 numbers and their positions correctly.)


Game Difficulties


Easy Mode

Easy Mode allows you to see a number if you correctly guess its position. For instance, if the secret number is 1234, and you guessed 5674, the game will show ***4 instead of ****. It's the default option and is recommended for beginners and casual experience.


Hard Mode

Hard Mode never lets you see the number until you actually win the game. It's recommended for players who're looking for a challenge.
